Get Data

Step 2a: Getting Data from Quandl.com

Go to quandl.com under For Everyone click on Browse All Core Data Feeds


Then Select Real Estate and when Rendered Click on Zillow Real Estate Research

Scroll to Dataset desired and click on "EXPAND"

At Upper right corner Under Export data Click on API "JSON"

Sign up to get data


Enter your First Name, Last Name , click on Personal and then Click on Next

Follow steps to complete the signup process.

Once your account is verified you can get the URL for API call that will be used to get data.


Note:  If this API is used outside of anything but training, please generate your own API KEY.
